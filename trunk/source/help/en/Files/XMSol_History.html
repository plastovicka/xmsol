What's new

Changes in version 1.6

  • Middle mouse button reveals card which is partially visible in a column
  • Settings can be saved as an INI file or in the registry
  • Added support for JPEG, PNG, GIF, TIFF picture formats (Windows XP or later)
  • The redeal seed (shuffle number) is now displayed in the status bar
  • The timer stops while a dialog window is open
  • The status bar can be hidden
  • Added a "Fast forward" (auto-play once) button in the tool bar
  • Long animations can be aborted by mouse click
  • A default profile is created the first time the game is launched
  • Fixed a bug where it was possible to confirm "no rules" and crash the game
  • The "best score" button is now disabled if the selected game type has never been played before
  • Added Next and Previous commands in the Cell menu
  • Game list window is resize-able
  • More decks and backgrounds
  • Changed the application icon
