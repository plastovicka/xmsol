Menus

Game

Select game...

Opens the game list window. The list contains several columns for statistics (game played, won, lost, score, ...). Select a rule with the left mouse button and click OK to start a new game. If the current game is not completed, the game is considered lost.

New game

Shuffle the cards, redistribute and reset the game using the same rules.

New game number...

Specify a shuffle number (the shuffle number of the current game is located in the status bar), redistribute and reset the game using the same rules.

Random game

Start a game using a randomly picked game from the list.

Change player profile...

By default, MXSolitaire will create a temporary profile called "DEFAULT", you can create or switch to another profile anytime. When restarting XMSolitaire, the previously used profile is reloaded.

Statistics...

Display miscellaneous statistics for the current profile.

Language

Change the interface language with the one selected in the list. You can add you own language file, see the Modification section to learn how.

Customize toolbar...

Customize toolbar buttons.

Keyboard shortcuts...

Customize keyboard shortcuts.

Options...

Open the advanced options window (more about it later).

Save settings to INI file

settings are saved in an INI file in the installation folder. It is useful if you have installed XM Solitaire on USB portable drive. Otherwise settings are saved into Windows' Registry. When switching mode, the previous setting container is cleaned (the registry is cleaned or the INI file is deleted).

Delete settings

Clean the registry or delete the INI file.

Exit

Saves the current game, close XMSolitaire and go back to Windows; next time XMSolitaire is started, the game will automatically be resumed.

Move

Undo

Cancel the previous move.

Redo

Repeat a canceled move.

Undo all

Go back to the game's initial state (restart without shuffling the cards nor restarting the timer). Note that going back to the initial state will not count as a "no play".

Redo all

Repeat every canceled move.

Fast forward

Force "autoplay" once, useful when you reach the end of the game and didn't enable autoplay.

View

Mirror left/right

Mirrors the layout left to right (for instance, if the stock is on the left by default, it will appear on the right when the mirror mode is enabled)

Mirror up/down

Mirrors the layout top to bottom (for instance, if the stock is on the top by default, it will appear on the bottom when the mirror mode is enabled)

Spread cards

Increase the space between two stacked cards in a pile.

Tighten cards

Decrease the space between two stacked cards in a pile.

Show toolbar

Display the toolbar (the bar under the menus).

Show statusbar

Display the statusbar (the bar way down XMSolitaire window).

Refresh

Read cards images from files and update the menu. Useful if you modified or added new files.

Cards

You can add your own cards images, see the Modification section to learn how.

Card back

You can add your own card back image, see the Modification section to learn how.

Cell

You can add your own empty cells images, see the Modification section to learn how.

Background

You can add your own backgrounds, see the Modification section to learn how.

Help

Help

Display this help.

Rules

Display current game's rules.

About...

Display version number and link to web site.
